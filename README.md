# Tic-Tac-Toe

Minimal implementation of a "Tic-Tac-Toe" app using React Native

## Setup and installation

    $ npm install -d

## Usage

    $ # Start webpack dev server
    $ npm start

Now point your browser to [http://localhost:3333](http://localhost:3333)
